## g_tk参数

登录后从WebBrowser控件的cookie里拿到p_skey字段，计算出g_tk

```c#
  	private int GetG_TK(string p_skey)
  	{
  		string e = p_skey;
  		var t = 5381;
  		for (int n = 0, r = e.Length; n < r; ++n)
  		{
  			t += (t << 5) + e[n];
  		}
  		return t & 0x7FFFFFFF;
  	}  
```
## 相册图片质量和格式

- 质量：从相册列表里拿到的图片url，有多种质量可选，直接修改url的相关参数。

  - /a/是最小图片
  - /m/是相册封面正常图片，当url里包含psb时才可用。
  - /b/是原图片（大概）

- 格式：也是修改url里的参数。

  - t=5是webp格式
  - d=0是jpge格式
